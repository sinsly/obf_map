## m40fuscation changelog

## [0.1.0] - Initial release | 11/7/25

* Project started.
* Supports embedding Lua scripts as `obf_string` + `rev_mapping`.
* Generates pasteable block from plaintext `src`.
* Loader reconstructs and executes embedded payload.
* Validation of embedded payload before execution.
* Security levels introduced 1 and 2
